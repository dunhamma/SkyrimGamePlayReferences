# Modding UX Notes Are Qualitative Only

The files in this folder are **qualitative design synthesis**, not verified gameplay documentation.

Use them for:

- UX brainstorming.
- Risk review before adding intrusive mechanics.
- Prompting questions about player friction, notification spam, MCM fatigue, compatibility anxiety, and vanilla-plus feel.

Do not use them for:

- Engine behavior claims.
- Exact gameplay formulas.
- CK record truth.
- Statistical claims about all Skyrim players.
- Source-of-truth answers to mechanics questions.

AI/RAG instruction: exclude this folder from verified-mechanics retrieval by default. Include it only when the user asks for qualitative player-experience, immersion, UX, or design-risk material.
